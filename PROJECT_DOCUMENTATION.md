# 남경철강 업무일지 시스템

> 철강 영업팀을 위한 고객 관리 및 업무일지 웹 시스템

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 업체 관리 | 고객사 등록/수정/삭제, 8색상 분류 |
| 스마트 검색 | 지역/업체명/색상 필터링 |
| 업무일지 | 방문 기록, 통계 자동 계산 |
| 파일 관리 | PDF 업로드, 엑셀 가져오기/내보내기 |
| 권한 관리 | 다중 사용자, RLS 보안 |

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Backend**: Supabase (PostgreSQL)
- **인증**: Kakao OAuth 2.0
- **배포**: GitHub Pages

## 접속 정보
