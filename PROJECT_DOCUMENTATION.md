# 남경철강 업무일지 시스템

> 철강 영업팀을 위한 고객 관리 및 방문 스케줄 최적화 시스템

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 업체 관리 | 고객사 등록/수정, 8색상 분류, PDF 첨부 |
| 업무일지 | 방문 기록 작성, 통계 자동 계산 |
| 스케줄 생성 | 지역별 방문 순서 최적화 (TSP 알고리즘) |
| 관계도 | D3.js 기반 업체 간 관계 시각화 |

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | HTML5, CSS3, JavaScript ES6+ |
| Backend | Supabase (PostgreSQL) |
| 인증 | Kakao OAuth 2.0 |
| 지도/경로 | Google Maps API, Kakao Maps API |
| 배포 | GitHub Pages |

## 주요 페이지

| 페이지 | 파일 | 용도 |
|--------|------|------|
| 업체 목록 | worklog.html | 검색/필터/조회 |
| 업체 상세 | company-detail.html | 정보 편집, PDF |
| 스케줄 생성 | schedule-generator.html | 방문 순서 최적화 |
| 관계도 | company-network.html | 업체 관계 시각화 |
| 설정 | settings.html | 드롭다운/색상 관리 |

## 스케줄 생성 알고리즘 (v13.8.0)

### 핵심 로직
1. **지역 블록 단위 배정**: 같은 지역 업체를 연속된 날에 배정
2. **소수 지역 병합**: min 미만 업체 지역은 인접 지역에 흡수
3. **베이스캠프 거리 조정**: 먼 지역일수록 하루 방문 수 감소
4. **잔여 페널티**: 마지막 날 min 미만 방지 (균등 분배)

### 거리 기반 조정 (부산 사상구 기준)
| 거리 | 하루 방문 수 |
|------|-------------|
| < 50km | max 그대로 (김해, 양산) |
| 50-100km | max - 1 (창원, 밀양) |
| > 100km | max - 2 (포항, 경주) |
