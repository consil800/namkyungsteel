# 사용자 가이드 👥

## 시스템 접속 및 로그인

### 1. 시스템 접속
- 웹사이트: `https://namkyungsteel.com`
- 지원 브라우저: Chrome, Firefox, Safari, Edge (최신 버전)

### 2. 로그인 방법
1. **카카오 로그인** 버튼 클릭
2. 카카오 계정으로 로그인
3. 최초 로그인 시 관리자 승인 대기
4. 승인 후 시스템 사용 가능

## 🏢 업체 관리

### 업체 목록 조회 (`worklog.html`)
- **지역별 검색**: 드롭다운에서 지역 선택
- **업체명 검색**: 검색창에 업체명 입력
- **정렬**: 컬럼 헤더 클릭으로 정렬 (↑↓ 화살표 표시)
- **색상 코딩**: 업체별 색상으로 상태 구분
- **전화 걸기**: 전화번호 클릭 시 바로 전화 연결

### 새 업체 등록 (`company-register.html`)
1. **필수 정보 입력**
   - 업체명, 지역 (필수)
   - 주소, 전화번호, 담당자 등
2. **업종 및 제품 정보**
   - 업종, 제조품, 사용품목
3. **색상 설정**
   - 업체 구분용 색상 선택
   - 색상별 의미는 설정에서 관리
4. **저장**: 모든 정보 입력 후 "등록" 버튼

### 업체 정보 수정 (`company-detail.html`)
1. 업체 목록에서 업체명 클릭
2. "업체 정보 수정" 버튼 클릭
3. 정보 수정 후 "수정" 버튼
4. **PDF 파일 업로드**:
   - 파일명은 영어/숫자로 작성
   - 여러 파일 동시 업로드 가능
   - 기존 파일 조회 및 삭제 가능

## 📝 업무일지 관리

### 업무일지 작성 (`work-log-entry.html`)
1. **기본 정보**
   - 방문 업체 선택 (자동완성)
   - 방문일, 방문목적
2. **방문 내용**
   - 면담자, 협의 내용
   - 다음 액션, 후속 일정
3. **저장**: 작성 완료 후 "저장" 버튼
4. **자동 업데이트**: 업체 방문횟수, 최근방문일 자동 갱신

### 업무일지 조회
- **업체별 조회**: 업체 상세 페이지에서 확인
- **전체 조회**: 대시보드에서 최근 활동 확인
- **수정/삭제**: 작성한 일지 수정 및 삭제 가능

## ⚙️ 개인 설정 (`settings.html`)

### 드롭다운 설정 관리
1. **결제조건**: 자주 사용하는 결제 조건 추가
2. **업종**: 업체 등록 시 사용할 업종 추가
3. **지역**: 업체 검색 시 사용할 지역 추가
4. **방문목적**: 업무일지 작성 시 사용할 목적 추가

### 색상 관리
1. **색상 추가**:
   - 색상명 입력 (예: "긴급 고객")
   - 색상 선택 (컬러 피커 사용)
   - 의미 입력 (예: "우선 대응 필요")
   - 방문일 숨김 옵션 (회색 고객용)
2. **색상 수정**: 기존 색상의 의미 변경
3. **색상 삭제**: 불필요한 색상 제거

### 색상 의미 가이드
- 설정한 색상별 의미가 자동으로 표시
- 업체 목록 및 상세 페이지에서 참고용으로 활용

## 🕸️ 업체 관계도 (`company-network.html`)

### 관계도 접근
1. 업체 상세 페이지에서 **"업체 관계도"** 버튼 클릭
2. 해당 업체가 중심이 되는 네트워크 차트 화면 이동
3. 인터랙티브 차트에서 업체 간 관계 시각화

### 기본 조작
- **드래그**: 업체 노드를 마우스로 끌어서 위치 이동
- **줌**: 마우스 휠로 차트 확대/축소
- **팬**: 차트 배경을 드래그하여 화면 이동
- **노드 클릭**: 관계 설정을 위한 업체 선택

### 업체 추가하기
1. **검색으로 추가**:
   - 좌상단 검색창에 업체명 입력
   - 등록된 업체 목록에서 선택
   - 자동으로 해당 업체 색상 적용
2. **직접 추가**:
   - 검색창에 새 업체명 입력
   - "직접 추가" 옵션 선택
   - 회색(임시 업체)으로 추가됨

### 관계 설정하기
1. **시작 업체** 노드 클릭 (왼쪽 관계 폼에 업체명 표시)
2. **대상 업체** 노드 클릭 (오른쪽 관계 폼에 업체명 표시)
3. **관계 유형** 선택 (납품, 매입, 자회사, 모회사 등)
4. **"관계 추가"** 버튼 클릭
5. 화살표가 있는 관계선 자동 생성

### 고급 기능
- **다른 업체로 이동**: 등록된 업체 더블클릭 시 해당 업체 중심 관계도로 이동
- **자동 배치**: "자동 배치" 버튼으로 노드들 재정렬
- **관계도 저장**: "저장" 버튼으로 현재 네트워크 저장
- **초기화**: "모두 삭제"로 중심 업체 외 모든 관계 제거

## 📊 대시보드 활용 (`employee-dashboard.html`)

### 주요 지표 확인
- **총 등록 업체 수**: 내가 등록한 업체 현황
- **이번 달 방문 현황**: 월별 활동 통계
- **최근 업무일지**: 최근 작성한 일지 목록

### 빠른 작업
- **새 업체 등록**: 대시보드에서 바로 이동
- **업무일지 작성**: 빠른 일지 작성 링크
- **업체 검색**: 검색창에서 즉시 검색

## 🔍 검색 및 필터링 팁

### 효율적인 검색 방법
1. **지역별 검색**: 드롭다운으로 지역 필터링
2. **업체명 부분 검색**: 전체 이름을 몰라도 일부만 입력
3. **검색 상태 유지**: 뒤로가기 시 검색 조건 유지됨
4. **정렬 활용**: 방문횟수, 최근방문일로 정렬하여 우선순위 파악

### 색상 코딩 활용
- **빨강**: 긴급/중요 고객
- **노랑**: 주의 필요 고객
- **초록**: 정상 거래 고객
- **회색**: 거래 중단 고객 (방문일 숨김)
- 색상별 의미는 설정에서 자유롭게 변경 가능

## 📱 모바일 사용법

### 모바일 최적화 기능
- **터치 친화적 UI**: 큰 버튼과 터치 영역
- **반응형 디자인**: 화면 크기에 따라 자동 조정
- **빠른 전화 연결**: 전화번호 터치로 즉시 통화

### 모바일에서 주의사항
- **파일 업로드**: PDF 파일명을 영어로 미리 변경
- **긴 텍스트**: 메모나 내용 입력 시 자동 저장 활용
- **네트워크**: 안정적인 인터넷 연결 상태에서 사용

## 🚨 자주 발생하는 문제

### 로그인 문제
- **해결**: 브라우저 캐시 삭제 후 재시도
- **카카오 인증 오류**: 다른 브라우저에서 시도

### 데이터 저장 문제
- **해결**: 필수 항목 (업체명, 지역) 입력 확인
- **네트워크 오류**: 잠시 후 다시 시도

### 파일 업로드 문제
- **PDF 파일명**: 한글 → 영어/숫자로 변경
- **파일 크기**: 50MB 이하로 제한
- **파일 형식**: PDF 파일만 업로드 가능
